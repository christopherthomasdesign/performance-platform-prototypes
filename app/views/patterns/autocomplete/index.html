Autocomplete

What is it?

The pattern is a list of suggestions based on what a user has already input. A user can navigate through the suggestions and selecting one adds the text in the input. Autocomplete can speed up the input of data and ensure the correct spelling is submitted. A common usage is for providing a country name.
